# Dataset Information

This project uses the **PAPILA subset of the publicly available SMDG-19 dataset** for multiclass
glaucoma stage classification from fundus images.

Due to licensing terms and privacy considerations, the dataset cannot be
redistributed within this repository.

## Dataset Access
Researchers may obtain the dataset from the original data providers:

Dataset link: https://www.kaggle.com/datasets/deathtrooper/multichannel-glaucoma-benchmark-dataset

After downloading and extracting the data, the dataset should be organized in
the following directory structure before running the code:
data/
└── PAPILA/
├── healthy/
├── glaucoma/
└── suspect/
