# Dataset Information

This project uses the **PAPILA subset of the publicly available SMDG-19 dataset** for multiclass
glaucoma stage classification from fundus images.

Due to licensing terms and privacy considerations, the dataset cannot be
redistributed within this repository.

## Dataset Access
Researchers may download dataset from the original data providers. Dataset Link: https://www.kaggle.com/datasets/deathtrooper/multichannel-glaucoma-benchmark-dataset.
The dataset should be organized in the following directory structure before running the code:
data/
└── PAPILA/
├── healthy/
├── glaucoma/
└── suspect/
